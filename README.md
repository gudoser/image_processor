# Image Processor (C++)

Консольное приложение для обработки изображений в формате BMP без использования сторонних библиотек.

## Возможности

- Чтение и запись 24-битных BMP изображений
- Последовательное применение фильтров через CLI
- Поддержка фильтров:
  - `crop` — обрезка изображения
  - `gs` — перевод в оттенки серого
  - `neg` — негатив
  - `sharp` — повышение резкости
  - `edge` — выделение границ
  - `blur` — гауссово размытие
  - `spiral` — закручивает изображение от центра

Фильтры применяются в порядке их указания.

---

## Пример использования

```bash
./image_processor input.bmp output.bmp -crop 800 600 -gs -blur 1.0
```

## Архитектура

Проект разделён на модули:

- `image/` — представление изображения и пикселей  
- `bmp/` — чтение и запись BMP  
- `filters/` — реализация фильтров  
- `controller/` — парсинг аргументов, последовательное применение фильтров  

Фильтры реализованы через общий интерфейс (`IFilter`) и применяются через pipeline.

---

## Особенности реализации

- Работа с бинарным форматом BMP:
  - чтение заголовков
  - обработка padding
  - валидация входных данных

- Алгоритмы обработки изображений:
  - свёртки (convolution)
  - Gaussian blur (separable kernel, оптимизация по времени)

- Надёжность:
  - обработка ошибок через исключения
  - проверка входных данных
  - защита от переполнений
  - покрытие ключевых компонентов юнит-тестами

- Работа с памятью:
  - RAII
  - отсутствие утечек памяти
  - отсутствие UB
